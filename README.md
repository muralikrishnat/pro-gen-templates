## 📖 Usage

1. **Browse** the templates above and find one that fits your needs.
2. **Copy** the template folder to your project directory:
   ```bash
   cp -r <folderPath> your-project-name
   ```
3. **Install** dependencies:
   ```bash
   cd your-project-name
   npm install
   ```
4. **Start** building!
